## 🎯 项目概述

### 功能特性

- ✅ **图片元数据读取** - 大小、格式、尺寸、修改时间

## 📖 详细使用方法

### 方法 1: 使用 local.bat（推荐）

**功能**: 完整分析（图片信息 + OCR）


## 🛠 故障排除

### 问题 1: "tesseract 不是内部或外部命令"

**原因**: 环境变量未配置或未生效

**解决方案**:
```cmd
# 使用完整路径运行
"C:\Program Files\Tesseract-OCR\tesseract.exe" --version

# 或重新打开命令行窗口
```

### 问题 2: "找不到语言包"

**原因**: 语言包文件缺失

**解决方案**:
```cmd
# 检查语言包
dir "C:\Program Files\Tesseract-OCR\tessdata\*.traineddata"

# 应该看到:
# chi_sim.traineddata
# eng.traineddata
# osd.traineddata
```

### 问题 3: "识别结果为空"

**原因**: 图片中没有文字或图片太小

**解决方案**:
1. 检查图片是否包含文字
2. 尝试放大图片
3. 确保图片清晰

### 问题 4: "识别结果乱码"

**原因**: 语言包不匹配或图片质量问题

**解决方案**:
1. 确保使用正确的语言包
2. 提高图片分辨率
3. 确保文字清晰可读

### 问题 5: "Node.js 版本过低"

**原因**: Node.js 版本不兼容

**解决方案**:
```cmd
# 检查版本
node --version

# 如果低于 v18，请升级
# 下载: https://nodejs.org/
```

### 问题 6: "权限错误"

**原因**: 没有读取文件的权限

**解决方案**:
1. 以管理员身份运行命令行
2. 检查文件是否被占用
3. 确认文件路径正确

## ❓ 常见问题

### Q1: 为什么选择 Tesseract？

**A**: Tesseract 的优势：
- ✅ 完全免费
- ✅ 离线运行
- ✅ 支持 100+ 语言
- ✅ 高精度识别
- ✅ 活跃的社区支持

### Q2: 支持哪些 Windows 版本？

**A**: 
- ✅ Windows 10
- ✅ Windows 11

### Q3: 可以识别手写文字吗？

**A**: 可以，但精度较低（约 70%）。建议：
- 使用清晰的手写图片
- 避免潦草字迹
- 提高图片分辨率

### Q4: 可以识别表格吗？

**A**: 可以识别表格中的文字，但不会保留表格结构。如需表格识别，建议：
- 使用专门的表格识别工具
- 或手动整理识别结果

### Q5: 可以批量处理多少张图片？

**A**: 理论上无限制，但建议：
- 单次批量处理不超过 100 张
- 大批量建议分批处理
- 注意磁盘空间和内存使用

### Q6: 识别速度如何？

**A**: 速度取决于：
- 图片大小：小图 < 1秒，大图 2-5秒
- 文字数量：文字越多越慢
- 硬件性能：CPU 越快越快

### Q7: 可以识别 PDF 吗？

**A**: 本工具仅支持图片格式。如需识别 PDF：
1. 先将 PDF 转换为图片
2. 再使用本工具识别

### Q8: 可以识别截图中的代码吗？

**A**: 可以识别代码文字，但：
- 不会保留代码格式
- 可能识别错误特殊符号
- 建议使用代码专用 OCR 工具

### Q9: 如何提高识别精度？

**A**: 建议：
1. 使用高分辨率图片
2. 确保文字清晰可读
3. 避免背景干扰
4. 使用合适的语言包
5. 图片预处理（去噪、增强对比度）

### Q10: 可以自定义识别语言吗？

**A**: 可以，修改 `analyze.js` 第 78 行：

```javascript
// 当前：中英文混合
const result = execSync(`"${tesseractPath}" "${imagePath}" stdout -l chi_sim+eng`, ...);

// 改为仅英文
const result = execSync(`"${tesseractPath}" "${imagePath}" stdout -l eng`, ...);

// 改为仅中文
const result = execSync(`"${tesseractPath}" "${imagePath}" stdout -l chi_sim`, ...);
```

## 🔍 技术细节

### 架构设计

```
用户输入 (图片路径)
    ↓
┌─────────────────┐
│  local.bat      │  命令入口
└────────┬────────┘
         ↓
┌─────────────────┐
│  analyze.js     │  主程序
├─────────────────┤
│ • 图片元数据    │  ← fs.statSync
│ • 尺寸获取      │  ← PowerShell
│ • 内容推断      │  ← 文件名分析
│ • OCR 识别      │  ← Tesseract
└────────┬────────┘
         ↓
    输出结果
```

### 依赖关系

```
F:\image-recognition\
├── local.bat           # 快捷入口
├── ocr.cmd             # OCR 快捷命令
├── analyze.js          # 主程序
│   ├── fs              # 文件系统
│   ├── path            # 路径处理
│   └── child_process   # 进程调用
└── node_modules/       # 依赖（无）
```

```javascript
// 完整路径调用
```
