# Mimo v2.5 Pro 图片识别解决方案

## 📋 问题描述

Mimo v2.5 Pro (Claude Code) **不支持直接显示图片**，即使：
- ✅ 已配置 Image MCP Server
- ✅ 使用 Read 工具读取图片
- ❌ 仍然显示 `[Unsupported Image]`

---

## 🎯 解决方案概述

### 两种方案任你选

| 方案 | 费用 | 配置 | 推荐度 |
|------|------|------|--------|
| **本地方案** | 免费 | 已完成 | ⭐⭐⭐⭐⭐ |
| **Claude API** | 付费 | 需配置 | ⭐⭐⭐⭐ |

### 方案 1: 本地方案（推荐）✅

**使用 Tesseract OCR，完全免费，无需 API Key**

```
本地图片 → Tesseract OCR → 文字内容 → Mimo 可以理解
```

**优点**:
- ✅ 完全免费
- ✅ 离线运行
- ✅ 无需配置
- ✅ 支持中英文

**已安装组件**:
- ✓ Tesseract OCR v5.4.0
- ✓ 中文语言包 (chi_sim)
- ✓ 英文语言包 (eng)

### 方案 2: Claude API 方案（备选）

**使用 Claude Vision API，需要 API Key**

```
本地图片 → Claude Vision API → 详细描述 → Mimo 可以理解
```

**优点**:
- ✅ AI 智能识别
- ✅ 详细描述
- ✅ 高精度

**需要配置**:
- ⚠️ Anthropic API Key
- ⚠️ 网络连接
- ⚠️ 付费使用

## 🚀 快速开始

### 本地方案（推荐）

**无需任何配置，直接使用！**

```cmd
# 打开命令行
cd F:\image-recognition

# 完整分析（图片信息 + OCR）
local.bat F:\screenshot.png

# 仅 OCR 文字识别
ocr.cmd F:\screenshot.png
```

### Claude API 方案

**需要先配置 API Key**

```cmd
# 步骤 1: 配置 API Key
set ANTHROPIC_API_KEY=sk-ant-xxxxx

# 步骤 2: 使用工具
cd F:\image-recognition
see.bat F:\screenshot.png
```

## 📖 详细使用方法

### 本地方案

#### 核心命令

```cmd
cd F:\image-recognition

# 完整分析（图片信息 + OCR）
local.bat F:\1.png

# 仅 OCR 识别
ocr.cmd F:\1.png

# 列出目录图片
local.bat --list F:\

# 批量分析
local.bat --batch F:\photos
```

#### 直接使用 Node.js

```cmd
# 完整分析
node analyze.js F:\screenshot.png

# 仅 OCR
node analyze.js --ocr F:\screenshot.png

# 列出图片
node analyze.js --list F:\

# 批量分析
node analyze.js --batch F:\photos

# Windows OCR（备选）
node windows-ocr.js F:\screenshot.png
```

### Claude API 方案

#### 核心命令

```cmd
cd F:\image-recognition

# 识别图片
see.bat F:\screenshot.png

# 自定义提示词
see.bat F:\screenshot.png "请提取图片中的文字"

# 列出图片
see.bat --list F:\

# 批量识别
see.bat --batch F:\photos
```

#### 直接使用 Node.js

```cmd
# 识别图片
node recognize.js F:\screenshot.png

# 自定义提示词
node recognize.js F:\screenshot.png "描述图片内容"

# 列出图片
node recognize.js --list F:\

# 批量识别
node recognize.js --batch F:\photos
```

## 💰 费用说明

### 本地方案

**完全免费**
- ✅ Tesseract OCR - 开源免费
- ✅ 语言包 - 免费下载
- ✅ Node.js 脚本 - 免费使用
- ✅ 离线运行 - 无网络费用

### Claude API 方案

**按使用量付费**
- 使用 Claude 3.5 Sonnet 模型
- 图片按 token 计费
- 平均每张图片约 $0.01 - $0.05

**估算费用**:

| 图片数量 | 预计费用 |
|---------|---------|
| 10 张 | $0.10 - $0.50 |
| 100 张 | $1.00 - $5.00 |
| 1000 张 | $10.00 - $50.00 |

**节省费用建议**:
1. 压缩图片大小
2. 使用更具体的提示词
3. 批量处理时先测试小样本
4. 避免重复识别同一图片

## 🛠 故障排除

### 本地方案问题

**问题 1: "tesseract 不是内部命令"**
```cmd
# 使用完整路径（已配置）
node analyze.js --ocr F:\image.png
```

**问题 2: "识别结果为空"**
- 检查图片是否包含文字
- 确保图片清晰
- 尝试放大图片

**问题 3: "识别结果乱码"**
- 确保语言包正确
- 提高图片分辨率
- 使用合适的语言包

### Claude API 方案问题

**问题 1: "API Key 无效"**
- 检查 Key 格式（以 `sk-ant-` 开头）
- 确认账户有余额
- 确认 Key 未过期

**问题 2: "文件不存在"**
- 使用绝对路径
- 确认文件存在
- 检查路径格式

**问题 3: "网络错误"**
- 检查网络连接
- 确认可以访问 api.anthropic.com
- 检查代理设置
