# Image Recognition Tool - 本地图片识别方案

## 📋 项目概述

**基于Tesseract OCR的完全免费、本地运行的图片识别工具**

### 核心功能

- ✅ **图片元数据** - 大小、格式、尺寸、修改时间
- ✅ **OCR 文字识别** - 支持中英文混合识别
- ✅ **批量处理** - 一次处理整个目录
- ✅ **完全免费** - 无需API Key，本地运行
- ✅ **隐私安全** - 数据不离开你的电脑

---

## 🎯 技术方案

### Tesseract OCR（唯一方案）

```
本地图片 → Tesseract OCR → 文字内容
```

**优势**:
- ✅ 完全免费
- ✅ 离线运行
- ✅ 无需API Key
- ✅ 无需网络连接
- ✅ 支持中英文
- ✅ 数据隐私安全

**已安装组件**:

- ✓ 中文语言包 (chi_sim)
- ✓ 英文语言包 (eng)

## 🚀 快速开始

### 无需任何配置，直接使用！


## 📖 详细使用方法

### 核心命令

```cmd
cd F:\image-recognition

# 完整分析（图片信息 + OCR）
local.bat F:\1.png

# 仅 OCR 识别
ocr.cmd F:\1.png

# 列出目录图片
local.bat --list F:\

# 批量分析
local.bat --batch F:\photos
```

### 直接使用 Node.js

```cmd
# 完整分析
node analyze.js F:\screenshot.png

# 仅 OCR
node analyze.js --ocr F:\screenshot.png

# 列出图片
node analyze.js --list F:\

# 批量分析
node analyze.js --batch F:\photos
```

### npm 脚本方式

```cmd
# 完整分析
npm run local F:\screenshot.png

# 仅 OCR
npm run local:ocr F:\screenshot.png

# 批量分析
npm run local:batch F:\photos
```

## 🚀 安装说明

### 一键安装（推荐）

```bash
# 方式1：快速安装
quick-install.bat

# 方式2：完整安装
install-ocr.bat

# 方式3：PowerShell安装
.\install.ps1
```

### 手动安装

1. 下载 Tesseract OCR：https://github.com/UB-Mannheim/tesseract/wiki
2. 安装到 `C:\Program Files\Tesseract-OCR\`
3. 下载语言包：chi_sim.traineddata, eng.traineddata
4. 复制到 `C:\Program Files\Tesseract-OCR\tessdata\`
