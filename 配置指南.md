# Image Recognition Tool - 配置指南

## 📋 概述

本项目提供两种图片识别方案：

1. **本地方案** ✅ 推荐 - 使用 Tesseract OCR，完全免费，无需 API Key
2. **Claude API 方案** - 使用 Claude Vision API，需要 API Key，付费

## 🎯 方案 2: Claude API 方案（备选）

### 配置步骤

#### 步骤 1: 获取 API Key

1. 访问 [Anthropic Console](https://console.anthropic.com/)
2. 登录/注册
3. 创建 API Key
4. 复制保存

#### 步骤 2: 配置 API Key

**方法 1: 环境变量（推荐）**

Windows PowerShell:
```powershell
$env:ANTHROPIC_API_KEY="sk-ant-xxxxx"
```

Windows CMD:
```cmd
set ANTHROPIC_API_KEY=sk-ant-xxxxx
```

**永久设置（Windows）:**
1. 右键"此电脑" → "属性"
2. "高级系统设置" → "环境变量"
3. 添加新的用户变量：
   - 变量名: `ANTHROPIC_API_KEY`
   - 变量值: `sk-ant-xxxxx`

**方法 2: 修改脚本**

编辑 `recognize.js`，修改第 6 行：
```javascript
const client = new Anthropic({
  apiKey: "sk-ant-xxxxx",  // 替换为你的 API Key
});
```

#### 步骤 3: 使用工具

```cmd
cd F:\image-recognition

# 使用快速启动脚本
see.bat F:\screenshot.png

# 或直接运行 Node
node recognize.js F:\screenshot.png
```

### 核心命令

| 命令 | 功能 | 示例 |
|------|------|------|
| `see.bat <路径>` | 识别图片 | `see.bat F:\1.png` |
| `see.bat <路径> "提示词"` | 自定义提示词 | `see.bat F:\1.png "提取文字"` |
| `see.bat --list <目录>` | 列出图片 | `see.bat --list F:\` |
| `see.bat --batch <目录>` | 批量识别 | `see.bat --batch F:\photos` |

## 🛠 故障排除

### 本地方案问题

**问题 1: "tesseract 不是内部命令"**
```cmd
# 使用完整路径（已配置）
node analyze.js --ocr F:\image.png
```

**问题 2: "识别结果为空"**
- 检查图片是否包含文字
- 确保图片清晰
- 尝试放大图片

**问题 3: "识别结果乱码"**
- 确保语言包正确
- 提高图片分辨率

### Claude API 问题

**问题 1: "API Key 无效"**
- 检查 Key 格式（以 `sk-ant-` 开头）
- 确认账户有余额
- 确认 Key 未过期

**问题 2: "文件不存在"**
- 使用绝对路径
- 确认文件存在
- 检查路径格式
