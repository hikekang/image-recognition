## 🚀 使用方法

### 核心命令

| 命令 | 功能 | 示例 |
|------|------|------|
| `local.bat <路径>` | 完整分析 | `local.bat F:\1.png` |
| `ocr.cmd <路径>` | OCR 识别 | `ocr.cmd F:\1.png` |
| `local.bat --list <目录>` | 列出图片 | `local.bat --list F:\` |
| `local.bat --batch <目录>` | 批量分析 | `local.bat --batch F:\photos` |

### 快速开始

```cmd
# 打开命令行，进入项目目录
cd F:\image-recognition

# 完整分析（图片信息 + OCR）
local.bat F:\screenshot.png

# 仅 OCR 文字识别
ocr.cmd F:\screenshot.png
```

## 📦 安装说明

### 一键安装（推荐）

```bash
# 方式1：快速安装
quick-install.bat

# 方式2：完整安装
install-ocr.bat

# 方式3：PowerShell安装
.\install.ps1
```

### 手动安装

1. **下载 Tesseract OCR**
   - 访问：https://github.com/UB-Mannheim/tesseract/wiki
   - 下载最新版本

2. **安装 Tesseract**
   - 运行安装程序
   - 安装到默认位置：`C:\Program Files\Tesseract-OCR\`

3. **下载语言包**
   - 中文：chi_sim.traineddata
   - 英文：eng.traineddata

4. **复制语言包**
   - 复制到：`C:\Program Files\Tesseract-OCR\tessdata\`

## 📝 详细使用

### 方法 1: 使用 local.bat（推荐）

**功能**: 完整分析（图片信息 + OCR）

```cmd
cd F:\image-recognition

# 分析单张图片
local.bat F:\1.png

# 分析带空格的路径
local.bat "F:\my images\photo.png"

# 列出目录图片
local.bat --list F:\

# 批量分析
local.bat --batch F:\photos
```

### 方法 2: 使用 ocr.cmd

**功能**: 仅 OCR 文字识别

```cmd
cd F:\image-recognition

# OCR 识别
ocr.cmd F:\screenshot.png

# 带空格的路径
ocr.cmd "F:\my images\screenshot.png"
```

### 方法 3: 直接使用 Node.js

```cmd
cd F:\image-recognition

# 完整分析
node analyze.js F:\screenshot.png

# 仅 OCR
node analyze.js --ocr F:\screenshot.png

# 列出图片
node analyze.js --list F:\

# 批量分析
node analyze.js --batch F:\photos
```

### 方法 4: 使用 npm 脚本

```cmd
# 完整分析
npm run local F:\screenshot.png

# 仅 OCR
npm run local:ocr F:\screenshot.png

# 批量分析
npm run local:batch F:\photos
```
